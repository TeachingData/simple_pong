# Simple Pong for testing GoDot

Originally made in GoDot v4.6.2 - this is a simple pong game for teaching the basics of game development.
It leaves many things to be desired (doesn't even keep score) as those are meant to be added by students.

## Based off GoDot docs
This is based off the [old GoDot docs](https://docs.godotengine.org/en/2.1/learning/step_by_step/simple_2d_game.html)
but updated (the original code doesn't work with 4.6 due to API changes).
